Proyecto WALL-E Controlado por Bluetooth

Explora el futuro de la robótica con este diseño inspirado en WALL-E: control inalámbrico, ingeniería creativa y tecnología inteligente.

Sobre el Proyecto

Este proyecto busca replicar el encanto de WALL-E mientras enseña los fundamentos de la robótica moderna. Mediante un módulo Bluetooth HC-05, el robot puede ser controlado de forma inalámbrica desde una aplicación móvil. Se han integrado motores de tracción, servos de rotación y una estructura diseñada para soportar diferentes terrenos.

El cerebro del robot es un microcontrolador Arduino Nano, encargado de coordinar cada movimiento con precisión. Gracias a su bajo consumo energético y su programación flexible, WALL-E puede realizar giros, avanzar, retroceder y responder instantáneamente a las órdenes del usuario.

Futuras Mejoras

Las siguientes versiones incluirán sensores ultrasónicos para evitar obstáculos, un sistema de voz interactivo, cámaras con transmisión en tiempo real y reconocimiento de comandos desde aplicaciones móviles. Estas innovaciones convertirán a WALL-E en un robot más autónomo, funcional y educativo.
